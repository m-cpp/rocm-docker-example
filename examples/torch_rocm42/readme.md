# Torch with ROCm 4.2

## About

This is an example of using PyTorch-ROCm official docker.

## Host Environment

- OS: Ubuntu18.04
- GPU: AMD Vega56

## Versions in Container

- ROCm: 4.2
- PyTorch: 1.8.1
- Python: 3.6.13

## How to use

1. Install ROCm4.2 to your host.
1. Install docker to your host.
1. run `./run_docker.sh`
